## Run Locally

Clone the project

```bash
  git clone https://https://github.com/EastonA01/EastonA01.github.io.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

When ready to run a development server, use:

```bash
  npm run dev
```
